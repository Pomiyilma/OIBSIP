                                  LOGIN AUTHENTICATION SYSTEM (TASK 4) SETUP AND RUNNING INSTRUCTIONS:

This project requires Python and pip. Since it uses a virtual environment and a database, reviewers must follow these steps to run it locally:

1)  Clone the repo: 

    git clone https://github.com/Pomiyilma/OIBSIP.git
    cd OASIS-INTERNSHIP-PROJECTS/Login

2)  Activate Virtual Environment:

    cd flask-auth
    .\Scripts\Activate.ps1

3)  Install Dependencies which are the required python libraries: 

    pip install -r ../requirements.txt

4)  Run the Application: 

    python main.py

5)  Access the app: 

    navigate the web browser through the link provided when main.py is run. \
